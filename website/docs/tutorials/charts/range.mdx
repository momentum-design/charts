# Range Chart

Range chart support a fill area on the dataset object which can be used to create space between two datasets.

```jsx live noInline
const data = [
  ['Year', 'Unfilled', 'Filled'],
  ['2004', 1000, 400],
  ['2005', 2170, 5460],
  ['2006', 660, 2120],
  ['2008', 3630, 1340],
  ['2009', 3030, 740],
  ['2010', 330, 2640],
  ['2011', 2930, 5340],
  ['2012', 1030, 2240],
];
const initialOptions = {
  seriesOptions: {
    tension: 0.3,
  },
  valueAxes: [
    {
      position: 'left',
      title: 'Left Title',
    },
  ],
};
const Example = () =>
  getExample(initialOptions, data, ({ key, data, options, setOptions }) => {
    const Buttons = () => (
      <>
        <Button onClick={() => setOptions(initialOptions)}>Default</Button>
        <Button
          onClick={() =>
            setOptions(
              {
                seriesOptions: {
                  styleMapping: {
                    Filled: { valueAxisIndex: 1 },
                  },
                },
                valueAxes: [
                  {
                    position: 'left',
                    title: 'Left Title',
                  },
                  {
                    gridDisplay: false,
                    position: 'right',
                    title: 'Right Title',
                  },
                ],
              },
              false,
            )
          }
        >
          Multi Axis
        </Button>
        <Button
          onClick={() =>
            setOptions({
              seriesOptions: {
                tension: 0,
              },
            })
          }
        >
          No Tension
        </Button>
        <Button onClick={() => setOptions({ categoryAxis: { stacked: true }, valueAxes: [{ stacked: true }] })}>
          Stacked
        </Button>
        {buttonsRelatedToColor(setOptions)}
      </>
    );

    return (
      <WithActions buttons={<Buttons />}>
        <mdw-chart key={key} type="range" data={JSON.stringify(data)} options={JSON.stringify(options)}></mdw-chart>
      </WithActions>
    );
  });

render(<Example />);
```
