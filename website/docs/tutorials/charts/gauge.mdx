# Gauge Chart

Gauge chart is a data visualization chart, usually used to show the relative position of a single indicator or value within a fixed range.

It looks similar to a car dashboard and helps people quickly understand where a value falls within a predetermined range.

Gauge charts usually consist of a circular ruler (gauge) and an indicator (needle).

```jsx live
function showComponent() {
  const data = {
    scales: {
      Low: 30,
      Normal: 80,
      High: 10,
    },
    value: 65
  }
  
  const options = {
    innerRadius: '90%',
  };
  return <md-chart
      type="gauge"
      data={JSON.stringify(data)}
      options={JSON.stringify(options)}
    ></md-chart>;
}
```

## Type

type = 'gauge'

## Data

Supports data formats:

| Mode                  | Type                               | Example                                                                                                                  |
| --------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| Array                 | {scales: object; value: number}[]|[{scales: { Low: 30,Normal: 80,High: 10,},value: 65}]                                               |
| Object                | {scales: object; value: number}  | {scales: { Low: 30,Normal: 80,High: 10,},value: 65}|

## Options

| Name   | Type   | Default | Description                                             |
| ------ | ------ | ------- | ------------------------------------------------------- |
| innerRadius | string | 90%     | The portion of the chart that is cut out of the middle. |
| padding | number \| { top: number;right: number;bottom: number;left: number; } | {left: 30,right: 30} | Padding values in Chart options can be supplied in a couple of different formats.|
| font | {family: string; color: string;} | {family: 'CiscoSansTT Regular,Helvetica Neue,Helvetica,Arial,sans-serif', color: '#000000'}     | The font and color of the scale |
