# Mixed Bar/Line Chart

Mixed Bar/Line Chart is a data visualization that combines the characteristics of both bar chart and line chart.

```jsx live
function showComponent() {
  const data = [
    {
      date: '2016',
      income: 23.5,
      expenses: 21.1,
    },
    {
      date: '2017',
      income: 26.2,
      expenses: 30.5,
    },
    {
      date: '2018',
      income: 30.1,
      expenses: 34.9,
    },
    {
      date: '2019',
      income: 29.5,
      expenses: 31.1,
    },
    {
      date: '2020',
      income: 30.6,
      expenses: 28.2,
    },
    {
      date: '2021',
      income: 34.1,
      expenses: 32.9,
    },
    {
      date: '2022',
      income: 36.1,
      expenses: 35.9,
    },
    {
      date: '2023',
      income: 24.1,
      expenses: 22.9,
    },
  ];
  const options = {
    categoryAxis: {
      gridDisplay: true,
      dataKey: 'date',
    },
    seriesOptions: {
      styleMapping: {
        income: {
          type: 'bar',
          order: 1,
        },
        expenses: {
          type: 'dashed',
          tension: 0.3,
          order: 0,
        },
      },
    },
  };
  return (
    <div>
      <md-chart type="line" data={JSON.stringify(data)} options={JSON.stringify(options)}></md-chart>
    </div>
  );
}
```
